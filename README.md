# Pokemon Champions Battle Desk

Pokemon Champions 싱글 배틀을 위한 비공식 포켓몬 비교 도우미입니다. 브라우저에서 바로 실행되며, 좌우 패널에서 포켓몬 두 마리를 독립적으로 검색하고 비교할 수 있습니다.

## 주요 기능

- 독립 스크롤을 갖는 1:1 좌·우 스플릿 뷰
- 한글·영문 검색, 자동완성, 방향키 선택 및 Enter 검색
- 사용률 TOP 100 홈 화면과 메가진화 폼 묶음 표시
- 포챔스 기준 기본 능력치와 원본 종족값 참고 표기
- 방어 상성, 채용 기술 TOP 10의 공격 상성, 도구 채용률
- 성격·스피드 노력치·구애스카프·랭크 보정을 반영한 스피드 계산기
- 포켓몬별 대표 실전 샘플 5개와 능력치 막대 시각화
- 브라우저 뒤로가기/앞으로가기 상태 복원 및 메가진화 폼 바로가기

## 이용 방법

GitHub Pages 주소를 열어 사용합니다. 포켓몬 이름을 입력하거나, 홈 화면의 사용률 목록을 선택하세요. 각 패널은 서로 완전히 독립적으로 동작합니다.

## 데이터 출처

이 앱은 화면을 열 때 공개 API에서 최신 정보를 요청합니다. 배틀 데이터를 별도로 저장하거나 재배포하지 않습니다.

- 배틀 통계: [Pokemon Champions Battle Data](https://championsbattledata.com/)
- 포켓몬·기술·도구의 한글 현지화 및 보조 정보: [PokéAPI](https://pokeapi.co/)

Pokemon Champions Battle Data의 [API Rules](https://championsbattledata.com/api-rules/) 및 [License](https://championsbattledata.com/license)를 준수합니다. PokéAPI 데이터는 해당 프로젝트의 [BSD-3-Clause License](https://github.com/PokeAPI/pokeapi/blob/master/LICENSE.md)를 따릅니다.
